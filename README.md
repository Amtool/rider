===

`rider` 是基于 [Stylus](http://stylus-lang.com) 与后处理器、无侵入风格的 CSS 样式工具库。它提供了以下功能：

 * 样式初始化
 * 顺时针简写
 * 缓动函数
 * 图片
 * 排版
 * 形状
 * 动画
 * 单位转换
 * 响应式工具
 * Retina 支持
 * 辅助函数
 * ...

基于 `rider` 开发 **项目** 或 **UI 样式库**，能有效提高开发效率，延长代码生命周期。
